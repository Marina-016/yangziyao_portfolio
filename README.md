## 部署到Vercel

### 方法一：通过Vercel Dashboard（推荐）

1. **创建GitHub仓库**
   - 在GitHub上创建一个新仓库
   - 把 `index.html` 和 `ppt_images/` 文件夹推送到仓库

2. **连接Vercel**
   - 访问 [vercel.com](https://vercel.com)
   - 用GitHub账号登录
   - 点击「New Project」
   - 选择你的仓库
   - 点击「Deploy」

3. **完成！**
   - 部署成功后会获得一个 `.vercel.app` 的链接

### 方法二：使用Vercel CLI

1. **安装Vercel CLI**
   ```bash
   npm i -g vercel
   ```

2. **部署**
   ```bash
   cd 项目资料
   vercel
   ```

3. **跟着提示操作即可**
